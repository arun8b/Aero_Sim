# ARUN M D — Open-Source Simulation Portfolio

This repository is a GitHub Pages-ready portfolio for aerospace-focused simulation projects using:

- OpenFOAM
- SU2
- FEniCSx
- PyFR
- FiPy
- Dedalus
- Python scientific-computing libraries

## How to publish as GitHub Pages

1. Create a new GitHub repository, for example `aerosim-portfolio`.
2. Upload all files from this folder.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save. GitHub will provide a public Pages URL.

## Suggested repository naming

- `aerosim-portfolio`
